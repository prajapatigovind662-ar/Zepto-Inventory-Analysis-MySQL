# 🛒 Zepto Inventory & Revenue Analysis Using MySQL

## 📌 Project Overview
This project focuses on analyzing Zepto’s inventory dataset using SQL to generate meaningful business insights related to pricing, discounts, inventory management, stock availability, and revenue optimization.

The project demonstrates how SQL can transform raw e-commerce inventory data into actionable business insights for data-driven decision-making.

---

## 🎯 Objectives
- Perform data exploration and cleaning
- Analyze inventory and pricing trends
- Identify high revenue-generating categories
- Detect premium out-of-stock products
- Analyze discount patterns
- Apply advanced SQL concepts and window functions
- Generate business-focused recommendations

## 📊 Key Business Insights
- Cooking Essentials and Munchies generate the highest estimated revenue
- Fruits & Vegetables offer the highest average discounts
- Premium products contribute significantly to inventory value
- Bulk products dominate total stock weight
- Inventory distribution is concentrated in high-demand categories
